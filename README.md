# ⌨️ YOurType - Typing Speed Test

A simple and interactive Typing Speed Test web application built using **HTML, CSS, and JavaScript**.  
It helps users improve typing speed, accuracy, and consistency in a fun way.

---

## 🚀 Features

- 📝 Random paragraph generation
- ⏱️ 60-second timer
- ⚡ Real-time typing feedback (correct / incorrect characters)
- 📊 WPM (Words Per Minute) calculation
- 🎯 Accuracy calculation
- 🔄 Restart functionality
- 📱 Responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ How It Works

- A random paragraph is loaded on page start
- User starts typing → timer begins automatically
- Each character is checked:
  - ✅ Correct → green
  - ❌ Incorrect → red
- After 60 seconds:
  - WPM is calculated
  - Accuracy is calculated
  - Input is disabled
